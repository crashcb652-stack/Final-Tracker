# University-Scholarship-Scraper-Engine

## Project Overview
This engine automates the tracking and scraping of university scholarship data to help students 
find opportunities faster.

## Documentation Note
* **Main Script**: the core logic is located in final_tracker.py.
* **Demonstration**: This script is the one featured in the submitted video demonstrations

* ## Installation & Requirements
* To run this engine, you must have Python installed along with the libraries:
* **Selenium**: For browser automation.
* **Chrome WebDriver**: To allow Selenium to interface with the browser.

* Install the dependencies using pip:
* 'pip install selenium'
